# MTS Training Services website

This package is prepared for Vercel using Next.js.

## Deploy to Vercel

1. Create an account at Vercel.
2. Create a new project.
3. Import or upload this folder.
4. Leave the default build settings as they are.
5. Click Deploy.

## Notes

- The logo is stored in `public/logo.jpg`.
- The enquiry form is set to submit through FormSubmit to `colin@masseytraining.co.uk`.
- On first use of FormSubmit, you may need to confirm the email address they send a verification email to.
- To connect your own domain later, add it inside the Vercel project settings.
